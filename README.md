# 📄 LLM PDF Chatbot (RAG)

An LLM-powered PDF Chatbot that allows users to interact with PDF documents using natural language queries. The system uses Retrieval-Augmented Generation (RAG) to fetch relevant context and generate accurate answers.

---

## 🚀 Features
- Upload and process PDF documents
- Convert text into vector embeddings
- Perform semantic search using FAISS
- Generate context-aware responses using LLMs
- Simple and interactive UI with Streamlit

---

## 🛠️ Tech Stack
- Python
- LangChain
- FAISS
- Hugging Face Embeddings
- OpenAI API
- Streamlit
- PyPDF2

---

## 📦 Installation

```bash
git clone https://github.com/yourusername/pdf-chatbot.git
cd pdf-chatbot
pip install -r requirements.txt
streamlit run app.py
```
